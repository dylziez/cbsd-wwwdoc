Redmine jail

For installing Redmine cbsd jail just follow this steps: 
% cbsd repo action=get sources=img name=redmine
or 
% cbsd repo action=get sources=img name=redmine ver=9.2
If you hosted on another ( non 9.2 ) FreeBSD version.
After image downloading please set correct name and ip address of the jail and run a install script that asks you for passwords of MySQL superuser and redmine SQL user: 
% cbsd jconfig jname=redmine redmineinstall





Now Redmine ready to use. Do not forget to set astart=1 in cbsd jconfig jname=redmine if you want for starting jail automatically.

By the starting jail via 
% cbsd jstart redmine
a clean Redmine installation in the jail will be available for accessing via http://FQDN with default login and password: admin




For non-interactive and automatic deploying (for example you create hosting with lots of RM installation for customer) you can use: 
   % cbsd jconfig jname=redmine redmineinstall rootpw= rmpw= fqdn= redmine_lang=
